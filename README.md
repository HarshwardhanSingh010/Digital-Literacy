This repository is my Digital Literacy Portfolio, submitted as part of the CSE0001

each mapping to one module of the course. As a Student Digital Ambassador,
the goal is to demonstrate digital awareness, professional online presence,
collaboration skills, communication etiquette, and cybercrime awareness.

##  Module Summaries

### Task 1 — Digital Literacy Infographic (Module 1)
Created a one-page visual infographic using **Canva** covering: what digital literacy
means, safe internet practices, and professional online presence. The design uses
icons and a clean layout suitable for sharing with batchmates.

### Task 2 — Student Digital Portfolio (Module 2)
Set up professional profiles on **GitHub**(https://github.com/HarshwardhanSingh010), **LinkedIn**(https://www.linkedin.com/in/harshwardhan-singh-panwar-45b9ba26a/), and **Kaggle**(https://www.kaggle.com/harshwardhansingh010). Each platform
serves a different purpose GitHub for code, LinkedIn for professional networking,
and Kaggle for data science learning.

### Task 3 — Coding & Collaboration Platforms (Module 3)
Completed a beginner challenge on **HackerRank** and built a 5-question
**Digital Literacy Awareness Quiz** using Google Forms.
🔗 [Take the Digital Literacy Quiz](https://docs.google.com/forms/d/e/1FAIpQLSfXTTdP49r76TI5epONWeWiqGF0fkL1uantn2xqzmemMIuqeQ/viewform?usp=publish-editor) 

### Task 4 — Email Etiquette (Module 4)
Drafted two professional emails (extension request + internship interest) and created
a Social Media Do's and Don'ts checklist with 7 items in each category.

### Task 5 — Cybercrime Awareness (Module 5)
Researched UPI/Online Payment Fraud, wrote a realistic 250-word case study, and
created a 10-point prevention checklist including two UPI-specific safety tips and
reporting channels (cybercrime.gov.in | Helpline: 1930).
